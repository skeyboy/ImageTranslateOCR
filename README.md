# ImageTranslateOCR

Android 图片 OCR 文字识别 + 翻译 + 原图覆盖工具

## 功能
- 📷 选择相册图片
- 🔍 ML Kit 中文 OCR 识别
- 🌐 ML Kit 翻译（中→英）
- 🎨 OpenCV Inpaint 擦除原文字
- ✍️ Canvas 绘制翻译结果
- 💾 保存到相册

## 技术栈
| 模块 | 方案 |
|------|------|
| OCR | Google ML Kit (中文) |
| 翻译 | Google ML Kit Translation |
| 擦除 | OpenCV Inpaint (Telea) |
| UI | Kotlin + ViewBinding |
